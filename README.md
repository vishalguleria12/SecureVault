# SecureVault

A secure password vault and security management application built with React, TypeScript, Vite, Tailwind CSS, and shadcn-ui.

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

This project includes a `public/vercel.json` for Vercel deployment with SPA routing support.
